# 🎬 Movie Revenue Analysis

## 📌 Project Overview
This project analyzes factors influencing movie revenue using exploratory data analysis (EDA). The goal is to identify key drivers of box office success.

## 📊 Key Analyses
- Movies released per year
- Top 5 highest-grossing movies
- Budget vs Gross relationship
- Revenue trends over time
- Top production companies
- Genre-based revenue analysis

## 🔍 Key Insights
- High-budget movies tend to generate higher revenue, but success is not guaranteed
- A few production companies dominate the industry
- Action and Adventure genres perform best financially
- Audience ratings have limited impact on revenue

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📁 Files
- `movie_eda.ipynb` → Full analysis notebook
-  `movie.csv` -> Dataset

## 🚀 Conclusion
The analysis shows that movie success is primarily driven by budget, production companies, and genre, rather than ratings alone.
